🍽️ FoodOps Pro

Simulateur de Gestion de Restaurant - Launcher

🚀 Instructions :
1. Windows : Double-cliquez sur les fichiers .bat
2. Terminal : Copiez-collez les commandes ci-dessous
3. Prérequis : Python 3.11+ et pip install pyyaml pandas pytest

📋 Commandes Terminal

🍽️ python start_pro.py
👨‍🏫 python start_admin.py
🧪 python demo_pro.py
🎮 python -m src.foodops_pro.cli
💡 Conseils :
• Débutants : Commencez par "Démo Rapide"
• Étudiants : Utilisez "Jouer Pro"
• Professeurs : Configurez avec "Mode Admin"
• Rapide : "Version Classique" pour un test express
